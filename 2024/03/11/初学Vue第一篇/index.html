初学Vue第一篇

1.打开vscode

2.新建一个终端

3.输入vue create 项目名称（我这里是vue-case）

4.配置vue项目（我这里选三个自主配置，第一个vue3默认配置，第二个vue2默认配置）

5.键盘上下键和空格键勾选vueRouter 和 vuex

6.选择vue版本

7.键入y需要为生产中的索引回退设置适当的服务器

8.一律键入y

9.创建完毕

10.项目结构目录

11.项目配置package.json

12.输入cd 项目名称

13.输入npm run serve运行

14.运行成功
